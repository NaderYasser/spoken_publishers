### Installation

Clone repo:

```sh
git clone https://github.com/NaderYasser/spoken_publishers.git
cd spoken_publishers
```


Install the dependencies:

```sh
yarn install
```

or

```sh
npm install
```

### JSON Server

Start json-server:

```sh
npm run json:server
```
